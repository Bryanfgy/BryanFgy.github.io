TopMovies


TopMovies is a website that makes use of Php, html, Bootstrap, and SQL to function. It has functionality for adding, editing and deleting reviews from a database for each available movie on the website, a searchbar to search for any of the movie entries using keywords and has a simple account system that makes it so you need to log in to leave a user review on a movie, if not the options to do so will not be available.
